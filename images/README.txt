IMAGES FOLDER - SIYA DEVELOPERS WEBSITE
========================================

Place your image files in this folder.

Expected files:
  logo.png        → Your company logo (shown in header)
  banner.jpg      → Main hero/banner image (shown at top of page)
  gallery1.jpg    → Gallery image 1
  gallery2.jpg    → Gallery image 2
  gallery3.jpg    → Gallery image 3
  gallery4.jpg    → Gallery image 4

Supported formats: .jpg, .jpeg, .png, .webp

NOTE: If an image is missing, the website will still work —
      it will show a placeholder box in its place.
