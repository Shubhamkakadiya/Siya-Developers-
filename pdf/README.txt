PDF FOLDER - SIYA DEVELOPERS WEBSITE
======================================

Place your PDF brochure and documents in this folder.

Expected file:
  brochure.pdf    → Project brochure (downloaded when user clicks "Brochure" button)

You can also add:
  floor-plan.pdf
  price-list.pdf
  any other documents

NOTE: When a visitor fills the form and clicks "Download Now",
      the file  pdf/brochure.pdf  will be downloaded automatically.
      Make sure to name your brochure file exactly:  brochure.pdf
